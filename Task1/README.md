# DataScience Bootcamp - Python Practice Projects

A collection of simple Python practice programs created as part of a DataScience bootcamp.

## Projects

### Level 1 Tasks

#### 1. Reverse a String (`ReverseAString.py`)
A program that takes user input and reverses the string.

#### 2. Check Prime Number (`CheckPrime.py`)
A program that checks if a given number is prime.
- Efficient algorithm checking up to √n
- Handles edge cases (negative numbers, 0, 1, 2)

#### 3. Factorial Calculator (`Factorial.py`)
A program that computes the factorial of a non-negative integer.
- Implements both iterative and recursive approaches
- Input validation for negative numbers

### Level 2 Tasks

#### 4. Count Vowels (`No_of_vowels.py`)
A program that counts the number of vowels in a given string.

#### 5. Find Largest Number (`LargestNumber.py`)
A program that finds the largest number in a user-provided list.
- Accepts space-separated numbers
- Handles empty lists and invalid input

#### 6. Fibonacci Sequence (`FibonacciSequence.py`)
A program that generates the Fibonacci sequence up to N terms.
- Implements both iterative and recursive approaches
- Displays sequence with visual formatting

### Level 3 Tasks

#### 7. CLI Calculator (`Calc_using_functions.py`)
A simple CLI calculator that performs basic arithmetic operations:
- Addition
- Subtraction
- Multiplication
- Division (with division by zero protection)

#### 8. Tic-Tac-Toe Game (`TicTacToe.py`)
A complete CLI-based Tic-Tac-Toe (X/O) game for 2 players.
- Interactive board display
- Input validation for moves
- Win detection (rows, columns, diagonals)
- Draw detection
- Replay option

## Getting Started

### Prerequisites
- Python 3 installed on your system

### Running the Programs
Each program can be run independently using:
```bash
python filename.py
```

For example:
```bash
python CheckPrime.py
python TicTacToe.py
```
